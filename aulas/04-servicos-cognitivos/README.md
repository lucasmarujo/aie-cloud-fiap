# Aula 4 — Serviços Cognitivos & APIs

## Objetivos de aprendizagem

Ao final desta aula, você será capaz de:

- Consumir APIs cognitivas prontas: Speech-to-Text, Text-to-Speech, Vision, Language (sentimento, entidades).
- Decidir entre **APIs prontas** vs **modelos customizados** (custo, latência, controle, dados de treino).
- Integrar Azure AI Services com aplicações Python via SDK.
- Construir um pipeline simples: imagem → análise → resposta cognitiva.
- Aplicar segurança: chaves de API via KeyVault, rate limiting, content filtering.

---

## Conexão com o Quantum Commerce

Você integra **capacidades cognitivas** ao e-commerce da Quantum Commerce: busca por imagem (Vision), recomendação por voz (Speech), análise de sentimento de reviews (Language) e/ou descrição automática de produtos.
