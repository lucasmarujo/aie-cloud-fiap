# Aula 5 — MLOps na Nuvem

## Objetivos de aprendizagem

Ao final desta aula, você será capaz de:

- Explicar o ciclo de vida de um modelo de ML: dados → treino → registro → serving → monitoramento.
- Treinar um modelo simples no Azure ML Studio (compute gerenciado).
- Versionar **dados, código e modelos** com MLflow.
- Registrar experimentos e comparar runs.
- Publicar um modelo como API de predição (endpoint gerenciado ou Function).
- Conectar conceitos a LLMOps (analogias com prompt versioning, eval registries).

---

## Conexão com o Quantum Commerce

Você implementa um **pipeline de MLOps** para o modelo de recomendação de produtos da Quantum Commerce: treino reproduzível, versionamento, deploy do endpoint e monitoramento básico.
