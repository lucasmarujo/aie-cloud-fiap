# Aula 2 — Armazenamento & Bancos de Dados na Nuvem

## Objetivos de aprendizagem

Ao final desta aula, você será capaz de:

- Diferenciar tipos de armazenamento: objetos (Blob), arquivos, discos e tabelas.
- Comparar bancos relacionais IaaS vs PaaS/SaaS (Azure SQL, Cosmos DB, RDS-equivalentes).
- Entender quando usar bancos analíticos (Synapse, BigQuery, Redshift).
- Provisionar Blob Storage e Azure SQL via Terraform.
- Aplicar segurança em camada de dados: Managed Identity, Private Endpoints, criptografia em repouso.

---

## Conexão com o Quantum Commerce

Você provisiona o **storage e o banco de dados** da Quantum Commerce: catálogo de produtos (Cosmos DB ou Azure SQL), logs de transação (Blob Storage) e dados de clientes. Esse será o backbone de dados das aulas seguintes.
